Basic Lists

Two additional style classes are provided for both ordered lists and unordered lists.

Unordered Lists

Optional list styles for unordered lists include fc-bullet-circle and fc-bullet-square.

  • Unstyled unordered list item 1
  • Unstyled unordered list item 2
<ul>
  <li>Unstyled unordered list item 1</li>
  <li>Unstyled unordered list item 2</li>
</ul>
  • fc-bullet-circle unordered list item 1
  • fc-bullet-circle unordered list item 2
<ul class="fc-bullet-circle">
  <li>fc-bullet-circle unordered list item 1</li>
  <li>fc-bullet-circle unordered list item 2</li>
</ul>
  • fc-bullet square unordered list item 1
  • fc-bullet square unordered list item 2
<ul class="fc-bullet-square">
  <li>fc-bullet square unordered list item 1</li>
  <li>fc-bullet square unordered list item 2</li>
</ul>

Ordered Lists

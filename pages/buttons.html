Overview

Buttons are available in the colors primary, secondary, error, and muted, as well as white. For the themed colors there is both a filled and outlined version. There is also a flat variant for the filled buttons. These classes can be applied to <button> elements, input elements of types button, submit, and reset, as well as links.

Filled Buttons

Solid colored buttons can be styled with the fc-btn-{color} class. Defaults are only supplied with primary, secondary, muted, error, and white colors to maintain accessible contrast. 

<button class="fc-btn-primary">Primary</button>
<button class="fc-btn-secondary">Secondary</button>
<button class="fc-btn-muted">Muted</button>
<button class="fc-btn-error">Error</button>
<button class="fc-btn-white">White</button>

Flat Buttons

All filled buttons can have the fc-btn-flat class applied to remove their box-shadow. 

<button class="fc-btn-primary fc-btn-flat">Primary</button>
<button class="fc-btn-secondary fc-btn-flat">Secondary</button>
<button class="fc-btn-muted fc-btn-flat">Muted</button>
<button class="fc-btn-error fc-btn-flat">Error</button>
<button class="fc-btn-white fc-btn-flat">White</button>

Outlined Buttons

The primary, secondary, muted, and error buttons have alternative outlined versions. These can be used by adding the fc-btn-outline class. 

<button class="fc-btn-primary fc-btn-outline">Primary</button>
<button class="fc-btn-secondary fc-btn-outline">Secondary</button>
<button class="fc-btn-muted fc-btn-outline">Muted</button>
<button class="fc-btn-error fc-btn-outline">Error</button>
